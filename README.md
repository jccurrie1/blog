# Jared Currie's Blog

My personal blog where I write about technology, software development, and things I find interesting.

**Live site:** [jaredcurrie.com](https://jaredcurrie.com)

## About

This is my corner of the internet where I share thoughts, tutorials, and experiences from my journey in software development. Built with modern web technologies because I believe in using the right tools for the job.

## Tech Stack

- **Next.js 15** with App Router
- **React 19**
- **TypeScript**
- **Tailwind CSS** for styling
- **Markdown** for content (with gray-matter + remark)

## Running Locally

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it.

## Writing Posts

Blog posts live in the `_posts/` directory as Markdown files. Each post needs front matter with title, date, excerpt, and images.
